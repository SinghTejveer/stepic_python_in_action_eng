# stepic_python_in_action_eng

Code for [Stepic][1] course [Python in Action (eng)][2]

## Requirements

 - Python 3
 - Inside virtual environment: `pip install -r requirements.txt`

## Utils
### Tests

 - `invoke tests`

### Syntax Validation

 - `invoke syntax`

### New Solution

 - `invoke new 135` will create solution and test file templates for s135
